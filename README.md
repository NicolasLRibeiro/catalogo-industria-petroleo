## Catálogo de Vendas – Indústria do Petróleo

Projeto acadêmico desenvolvido no SENAI utilizando a linguagem C,
simulando um sistema de catálogo de vendas para o setor petrolífero.

## Objetivo
Simular um sistema de vendas com autenticação simples, seleção de
produtos e cálculo de pagamento voltado para a indústria do petróleo.

## Funcionalidades
- Cadastro e verificação de senha
- Menu interativo de produtos
- Cálculo do total da compra
- Opções de pagamento com desconto e parcelamento

## Tecnologias Utilizadas
- Linguagem C
- Lógica de programação
- Estruturas condicionais e de repetição
